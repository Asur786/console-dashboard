# AI for BI Console POC

## Objective

Build an Executive Summary dashboard in Databricks Apps.

## Filters

- Channel
- Category
- Retailer
- Country

## KPIs

- Dollar Sales
- Dollar Share
- Volume Share
- Distribution
- YoY Growth

## Behaviour

- Load filter values dynamically.
- Apply Filters executes SQL.
- SQL returns aggregated KPI values.
- KPI cards refresh without page reload.
- Keep architecture reusable for future Admin and User Preferences pages.

## Design

Modern enterprise analytics dashboard similar to Microsoft Fabric / Power BI.

Responsive.

Clean.

Minimal.

Professional.
